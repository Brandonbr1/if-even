### if-even

## what is it
it is a  more efficent version of [is-Even](https://github.com/samuelmarina/is-even/)

## install
` npm install  @brandonbr1/if-even`

## How to use it
```js
var ifeven = require('@Brandonbr1/is-even');

isEven(1); //false
isEven(2); //true
isEven(3); //false
// And so on
```
